## Mengkonversi File SCSS ke CSS (CSS Preprocessor)

### Menggunakan Gulp

**1. install gulp dependecy menggunakan npm (node package manager)**
Windows

``` 
$ npm install
```

Linux & Mac
 
```
$ sudo npm install
```

**2. Menjalankan Gulp**

```
$ gulp
```
